## Instalação
Clone o repositório.
~~~
git clone https://github.com/vian4dev/flexbox-twitter.git
~~~
Acesse o diretório do projeto.
~~~
cd flexbox-twitter
~~~
Abra a aplicação é execute o _index.html_ no seu navegador.
